Welcome to the KGA documentation!

Introduction

Welcome to the documentation of the KNVB Game Analyzer (KGB) dashboard. The KGB dashboard provides the user with the opportunity to analyze matches in great detail. …The different tabs of the dashboard, are meant to be used for different types of analysis and provide the user with possibilities.

Goals and purposes

This documentation will contain in-depth information about all the code that runs behind the dashboard. Documentation is given for all modules, functions, callbacks, classes and sometimes variables. Besides documenting the code, the documentation can also be consulted to find out how to contribute to the project. For this, the pages on How to document and Getting started with KGA are crucial. Furthermore, the documentation should give the developers more detailed knowledge on the decisions made and the code that has been written before them, the general outline can also give the user more insight in how the tool should be used.

Where to look

Before looking through the entire documentation, carefully try to pinpoint what it is that you are looking for. For example: In what python file is the problem I am looking for located? What am I trying to learn? On what topics do I need to obtain more information? These questions help one to move more swiftly through this documentation. The side bar, the Index page, the Module index page, and the Search bar can be used to find specific topics in the documentation.
